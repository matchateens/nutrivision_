# 🇮🇩 NutriVision ID - National Nutrition Intelligence

**NutriVision ID** adalah platform dashboard intelijen gizi nasional yang dirancang untuk memantau prevalensi stunting dan mensimulasikan dampak program **Makan Bergizi Gratis (MBG)** di seluruh Indonesia.

Project ini dibangun untuk kompetisi **#juaravibecoding** dengan fokus pada estetika premium, interaktivitas tinggi, dan akurasi data berbasis SKI 2023.

---

## 🚀 Fitur Utama
- **Geospatial Analytics**: Peta interaktif 38 Provinsi Indonesia dengan kemampuan *drill-down* otomatis ke tingkat Kabupaten/Kota.
- **Strategic KPI Monitoring**: Ringkasan data gizi nasional (Stunting, Anak Terdampak, Target RPJMN 2029, & Penerima MBG) secara real-time.
- **MBG Simulator**: Alat proyeksi anggaran pintar yang memungkinkan pengguna mensimulasikan cakupan program MBG berdasarkan alokasi dana wilayah.
- **Intelligent Search**: Navigasi cepat melalui Search Bar untuk menemukan data gizi spesifik di setiap provinsi.
- **Responsive Design**: Optimalisasi layout untuk pengalaman terbaik di perangkat mobile (HP), tablet, hingga desktop.

---

## 🛠️ Tech Stack
- **Frontend**: React.js (Vite)
- **Styling**: Tailwind CSS (Glassmorphism UI)
- **Animations**: Framer Motion
- **Data Visualization**: Recharts & D3-Geo
- **Icons**: Lucide React
- **Maps**: React Simple Maps (TopoJSON)

---

## 📦 Instalasi & Cara Menjalankan

1. **Clone Repository**
   ```bash
   git clone https://github.com/username/nutrivision-id.git
   cd nutrivision-id
   ```

2. **Install Dependensi**
   ```bash
   npm install
   ```

3. **Jalankan Development Server**
   ```bash
   npm run dev
   ```
   Aplikasi akan berjalan di `http://localhost:5173`.

---

## 📊 Sumber Data
- **Stunting**: Survei Kesehatan Indonesia (SKI) 2023 / Kemenkes RI.
- **Populasi**: BPS (Badan Pusat Statistik) Sensus 2020 & Proyeksi 2023.
- **MBG**: Estimasi operasional Badan Gizi Nasional (BGN) 2025.
