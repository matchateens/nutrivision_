# Product Requirements Document (PRD): NutriVision ID

**Project Name**: NutriVision ID  
**Tagline**: National Nutrition Intelligence & MBG Simulator  
**Domain**: Health-Tech / Public Policy / Data Intelligence  
**Competition**: #juaravibecoding  

---

## 1. Executive Summary
Indonesia menghadapi tantangan besar dalam mencapai target prevalensi stunting 14% pada tahun 2029. Dengan diperkenalkannya program Makan Bergizi Gratis (MBG), diperlukan alat pemantauan yang transparan dan berbasis data untuk membantu pengambilan keputusan di tingkat lokal dan nasional. **NutriVision ID** hadir sebagai solusi visual yang menghubungkan data kesehatan dengan proyeksi fiskal secara interaktif.

## 2. Problem Statement
- **Fragmentasi Data**: Sulitnya membandingkan data gizi antar wilayah secara cepat.
- **Kekurangan Alat Simulasi**: Pembuat kebijakan kesulitan memproyeksikan cakupan program MBG berdasarkan keterbatasan anggaran.
- **Visualisasi yang Kaku**: Dashboard pemerintah yang ada seringkali sulit diakses via perangkat mobile dan kurang intuitif bagi publik.

## 3. Goals & Objectives
1.  **Transparency**: Menyediakan akses mudah terhadap data prevalensi stunting SKI 2023.
2.  **Actionable Insights**: Memungkinkan simulasi anggaran yang langsung menunjukkan estimasi penurunan stunting.
3.  **Accessibility**: Memberikan pengalaman dashboard "Intelligence" kelas atas yang responsif di semua perangkat.

## 4. User Persona
- **Policy Maker (Bappenas/Pemda)**: Membutuhkan data cepat untuk menentukan prioritas intervensi.
- **Researchers/Citizens**: Memantau perkembangan gizi di wilayah mereka.
- **Field Officers (Badan Gizi Nasional)**: Memeriksa kesiapan infrastruktur dapur (SPPG) per wilayah.

## 5. Functional Requirements
### Core Features:
- **Interactive Heatmap**: Peta Indonesia dengan pewarnaan otomatis (Aman/Waspada/Darurat).
- **Auto-Zoom Drill-Down**: Transisi mulus dari Nasional ke tingkat Provinsi dan Kabupaten.
- **Smart Search Bar**: Pencarian cerdas untuk akses langsung ke wilayah tertentu.
- **MBG Simulator Engine**: Algoritma penghitung cakupan anak, unit dapur dibutuhkan, dan proyeksi penurunan stunting tahunan.
- **Regional Analytics**: Grafik tren historis stunting (2019-2024) per wilayah.

## 6. Non-Functional Requirements
- **Performance**: Pemuatan peta di bawah 2 detik.
- **Aesthetics**: Desain Dark Mode "Premium Intelligence" dengan aksen Emerald.
- **Responsiveness**: Layout adaptif untuk Mobile (Grid 2x2 Header) dan Desktop (Multi-Column).

## 7. Success Metrics
- **User Engagement**: Kemudahan navigasi melalui fitur Search dan Map click.
- **Simulated Scenarios**: Berapa banyak skenario anggaran yang dapat dihasilkan pengguna melalui simulator.
